# Deepfake Detection 2026 With Random Forests

<image src="./dataset-cover.png" alt="Deepfake dataset thumbnail." />

This repository contains a hybrid machine learning pipeline designed to detect synthetic media using the **Kaggle Deepfake Detection 2026** dataset. The project utilizes a combination of **Deep Learning** for spatial feature extraction and **Ensemble Learning** for final classification.

---

## Project Overview

The core objective is to identify subtle artifacts in facial imagery generated by GANs or diffusion models. By leveraging a pre-trained **Xception** backbone, we extract high-dimensional feature vectors which are then processed by a **Random Forest Classifier** to determine authenticity.

### Key Performance

- **Accuracy:** ~99% on the test set.
- **Precision (Real):** 1.00 (Minimal false positives).
- **Recall (Fake):** 1.00 (High sensitivity to synthetic content).

## Architecture & Pipeline

The detection pipeline follows a multi-stage approach to ensure computational efficiency and high accuracy.

### 1. Preprocessing & Face Extraction

- **MTCNN Detection:** Automatically locates facial bounding boxes in raw imagery.
- **20% Margin:** Coordinates are expanded by a 20% margin to capture peripheral artifacts near the hairline and jaw.
- **Normalization:** Images are resized to **299x299** and normalized to the range expected by the Xception model.

### 2. Feature Extraction (The "Filter" Stage)

Instead of training a full Convolutional Neural Network (CNN) from scratch, we use **Transfer Learning**:

- **Backbone:** Xception (pre-trained on ImageNet).
- **GlobalAveragePooling2D:** This layer flattens the 4D feature maps into a **2048-dimensional** vector, summarizing the spatial "filters" detected by the network.

### 3. Classification

- **Random Forest Classifier:** A forest of 100 decision trees is trained on the extracted 2048 features.
- **Benefits:** This hybrid approach captures non-linear interactions between CNN features while being less prone to the vanishing gradient issues of fully connected deep layers.

## Installation & Usage

### Prerequisites

Ensure you have the following libraries installed:

```bash
pip install tensorflow keras scikit-learn mtcnn lz4 pandas numpy

```

### Local Execution

1. **Extract Features:** Run the extraction script to convert raw URLs/Images into `Xfeatures_train.csv`.
2. **Train Model:** Load the CSV into the Random Forest training script.
3. **Evaluate:** Use the classification report and confusion matrix to analyze performance.
